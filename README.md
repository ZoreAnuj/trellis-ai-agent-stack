# Trellis AI Agent Stack

A production-ready Next.js template for building AI agents with LangGraph.js. This project explores scalable agent architectures by providing a foundation for dynamic tool integration and persistent conversation management. It serves as a practical implementation for developing and deploying interactive AI agent systems.

## Key Features
*   **MCP Integration:** Dynamically loads tools at runtime using the Model Context Protocol.
*   **Human-in-the-Loop:** Implements user approval workflows for agent tool execution.
*   **Persistent Memory:** Maintains conversation history across sessions for coherent interactions.
*   **Full-Stack Agent:** Combines a LangGraph.js agent backend with a modern Next.js frontend.

## Tech Stack
*   Next.js 14 (App Router)
*   LangGraph.js / LangChain
*   TypeScript
*   Tailwind CSS

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/trellis-ai-agent-stack.git`
2.  Install dependencies: `npm install`
3.  Set up your environment variables in `.env.local`.
4.  Run the development server: `npm run dev`
